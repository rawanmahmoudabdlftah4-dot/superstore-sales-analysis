# superstore-sales-analysis
End-to-End Sales Analysis using Excel, Power Query, Pivot Tables, and Interactive Dashboard.
 Project Overview

This project analyzes sales performance, customer behavior, product profitability, and market trends using the Global Superstore dataset. The analysis aims to uncover actionable business insights and support data-driven decision-making.

 Tools Used
Microsoft Excel
Power Query
Pivot Tables
